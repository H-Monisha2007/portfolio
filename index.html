Credit Card Fraud Detection

ML detection on imbalanced datasets using LR & RF.

Problem Statement

Credit card transactions are vulnerable to fraudulent activities in online systems. Fraud cases are very rare compared to genuine transactions, causing data imbalance. This imbalance makes accurate fraud detection difficult for traditional methods. An automated and reliable detection system is required to identify fraud early.

Solution

The dataset is preprocessed, normalized, and balanced using undersampling techniques. Machine learning models like Logistic Regression and Random Forest are trained. Model performance is evaluated using classification reports, confusion matrix, and ROC curve. Random Forest provides better fraud detection accuracy and reliability.
